## About The Project

 <img src="img/captura.png" alt="Logo" width="100%" height="100%">


---
 Um projeto Full-Stack desenvolvido para demonstrar minhas habilidades em desenvolvimento web. Utilizando Java e Angular, esta aplicação oferece uma solução completa para criar e gerenciar uma loja virtual, 
 com recursos de gestão de produtos, carrinho de compras, integração de sistemas de pagamento, nota fiscal e frete para todo país. 
 Esta aplicação representa uma abordagem moderna e eficaz para o comércio eletrônico.
